Triage Intelligence

Three-level urgency classification

MIA reads the AI's response and automatically assigns an urgency level so nurses can prioritize at a glance.

🔴 HIGH URGENCY

Immediate physician notification required. Red-flag symptoms detected — potential life-threatening condition.

Examples: chest pain + radiation, unresponsive patient, signs of stroke or anaphylaxis

🟡 MODERATE URGENCY

Prompt assessment needed. Patient is stable but requires timely evaluation and nursing action.

Examples: high fever, moderate abdominal pain, dizziness, suspected fracture

🟢 LOW URGENCY

Non-urgent. Patient can be managed in standard queue with routine nursing assessment.

Examples: minor laceration, mild headache, prescription refill, routine follow-up

Quick Start

Up and running in 6 steps

No cloud account, no API keys, no Docker. Just Python and Ollama.

1

Install Ollama

Download and install Ollama for your operating system.

# Windows / macOS: download from
https://ollama.com/download

# Linux:
curl -fsSL https://ollama.com/install.sh | sh
2

Pull a model and register it as mia

Any Ollama model works. This example uses LLaMA 3.2 3B (fast, 2 GB).

ollama pull llama3.2:3b
echo "FROM llama3.2:3b" > Modelfile
ollama create mia -f Modelfile
3

Clone the repository

git clone https://github.com/DrFarooqAi/medical-agent.git
cd medical-agent
4

Install Python dependencies

pip install -r requirements.txt
5

Run the app

python app.py
6

Open in your browser

Navigate to the local address shown in your terminal.

http://localhost:7870
